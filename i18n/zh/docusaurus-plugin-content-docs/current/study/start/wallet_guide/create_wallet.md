# 如何获取并创建钱包

## 移动钱包还是桌面钱包？

移动设备钱包和桌面钱包各有优缺点。请考虑哪种钱包更符合您的需求，以便决定使用哪种类型的钱包。

| | 移动 | 桌面 |
| --------------------------------- | ------ | ------- |
| 随时随地使用 | ✅ | ➖ |
| 易于使用 | ✅ | ➖ |
| 更安全 | ➖ | ✅ |
| 无障碍访问 | ➖ | ✅ |
| 防损坏/丢失/盗窃 | ➖ | ✅ |
| 防电源/连接中断 | ✅ | ➖ |

## 如何获取？
